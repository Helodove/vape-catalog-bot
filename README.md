# Telegram-бот каталога вейп-шопа

Бот показывает каталог товаров из МойСклад прямо в Telegram. Покупатели могут листать категории, смотреть карточки товаров с фото и ценами, а также искать товары по названию.

---

## 1. Получить TELEGRAM_BOT_TOKEN

1. Откройте Telegram и найдите бота **@BotFather**.
2. Напишите `/newbot` и следуйте инструкциям.
3. В конце BotFather выдаст токен вида `1234567890:ABCdef...` — это и есть `TELEGRAM_BOT_TOKEN`.

---

## 2. Получить MOYSKLAD_TOKEN

1. Зайдите в МойСклад → **Настройки** (шестерёнка справа вверху).
2. Перейдите: **Доступ** → **Токены доступа**.
3. Нажмите **Создать токен**, скопируйте его — это `MOYSKLAD_TOKEN`.

---

## 3. Заполнить .env

Скопируйте `.env.example` в `.env` и заполните значения:

```
TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather
MOYSKLAD_TOKEN=ваш_токен_из_мойсклад
ADMIN_CHAT_ID=ваш_telegram_id
```

---

## 4. Найти свой ADMIN_CHAT_ID

Напишите боту **@userinfobot** в Telegram. Он ответит вашим числовым ID — вставьте его в `ADMIN_CHAT_ID`.

---

## 5. Запуск локально

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить бота
python bot.py
```

---

## 6. Запуск через Docker

```bash
# Собрать образ
docker build -t vape-catalog-bot .

# Запустить контейнер
docker run --env-file .env vape-catalog-bot
```

---

## Команды бота

| Команда    | Описание                                      |
|------------|-----------------------------------------------|
| `/start`   | Приветствие и главное меню                    |
| `/catalog` | Открыть каталог категорий                     |
| `/search`  | Поиск товара по названию                      |
| `/refresh` | Сбросить кэш (только для администратора)      |
